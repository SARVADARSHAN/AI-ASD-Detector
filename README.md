# 🚌 RouteRiders

RouteRiders is a modern Bus Booking System that allows users to search buses, book tickets, manage reservations, and view booking history through a clean and responsive interface.

It is built using a Flask backend, MySQL database, and a responsive HTML, CSS, and JavaScript frontend.

---

## 📌 Features

### 👤 User Features
- User Registration
- Secure Login & Logout
- Search buses by source and destination
- View available buses
- Book bus tickets
- Booking confirmation
- View booking history
- Responsive UI

### 🛠 Admin Features
- Add new buses
- Update bus details
- Delete buses
- Manage routes
- Manage bookings
- View registered users
